# ERIS

Election Roll Intelligence System

## Features

- Duplicate EPIC Detection
- Roll Comparison
- Address Analytics
- OCR Support
- AI Review
- Reports

## Tech Stack

Backend

- FastAPI
- SQLAlchemy

Frontend

- React
- Vite
- Electron

Database

- SQLite

## Run

Backend

```bash
cd backend
uvicorn app.main:app --reload
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Electron

```bash
npm run electron
```
